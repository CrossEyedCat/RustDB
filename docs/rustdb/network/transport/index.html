Shared QUIC [quinn::TransportConfig] for RustDB server and clients.

Keeps flow-control and stream limits aligned so the application semaphore in crate::network::query_stream::run_connection_streams is not stricter than the peer’s QUIC advertised limits (both are derived from the same max_concurrent_streams_per_connection).
