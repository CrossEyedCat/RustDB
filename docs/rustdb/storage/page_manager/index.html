MAX_RECORDS_PER_PAGE 🔒
Maximum records per page before bincode serialization exceeds PAGE_SIZE. CompactPage: header(~50) + slots(8 + n*17) + data(8 + ~12n) ≈ 130 + 29n. For PAGE_SIZE=4096: n ≤ 136. Use 100 for safety with variable record sizes.
