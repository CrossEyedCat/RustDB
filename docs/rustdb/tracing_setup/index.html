Global tracing subscriber for the RustDB server (tokio-rs/tracing).

Configures tracing-subscriber with:

  • [EnvFilter] from RUST_LOG (default info);
  • optional tracing-chrome JSON when RUSTDB_TRACE_CHROME_PATH is set (open in Chrome’s chrome://tracing);
  • fmt for human-readable stderr logs.

After the subscriber is installed, tracing-log bridges the legacy log crate so log::info! etc. are recorded by the same pipeline.
